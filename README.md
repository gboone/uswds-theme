== Description ==
A theme based on the U.S. Web Design Standards. Learn more at https://playbook.cio.gov/designstandards/

== Installation ==

1. Clone this repo to your wp-content/themes directory or [download this theme as a ZIP file](https://github.com/gboone/uswds-theme/archive/master.zip)
2. If you downloaded the zip file, log into your WordPress Admin and navigate to Appearance > themes
3. Click "Add new"
4. Click "Upload theme" and drag the ZIP file from your computer onto the "Choose File" button
5. Click the "Install now" button
